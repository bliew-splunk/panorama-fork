Panorama Release Notes - Published by Splunk January 27, 2022

Version 3.0.6 - Released January 27, 2022
  • Marked the app as FIPS Compliant [PAPP-22698]
Version 3.0.2 - Released September 1, 2021
  • Added support for the API changes in v10.1 [PAPP-18743]
Version 2.0.0 - Released March 19, 2021
  • Compatibility changes for Python 3 support
  • Exception handling for errors returned from APIs
  • Updated the app documentation
  • Updated the app logo
Version 1.1.6 - Released February 06, 2018
  • App action views and Logo updates
Version 1.1.4 - Released August 23, 2017
  • 3.0 compatibility update
Version 1.1.2 - Released August 1, 2017
  • Added "run query" action

  • Updated data paths
  • Updated documentation
Version 1.0.21 - Released December 12, 2016
  • Added an optional is_source_address parameter, to allow blocking source IPs as well as destination
Version 1.0.13 - Released October 14, 2016
  • Made improvements to the datapath settings
Version 1.0.12 - Released August 15, 2016
  • Fixed a datapath setting that could cause nulls to be returned
